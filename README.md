# Project Overview
SpareLot is a peer-to-peer storage and parking marketplace that connects people searching for storage to those with unused space. Renters can search for storage for any type of item or vehicle and hosts can rent out almost any space from basements to garages to driveways, and more. 

# Tech Stack
Frontend: ReactJS, HTML
Backend: Firebase, Node.js
Content Delivery: ImgBB
